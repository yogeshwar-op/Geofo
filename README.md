# ✈️ GeoFO

**GeoFO** is an open-source virtual First Officer for **GeoFS** that brings airline-style cockpit assistance to the simulator.

GeoFO acts as a professional First Officer, helping pilots with checklists, callouts, flight information, and aircraft monitoring through voice interaction.

> **Project Status:** 🚧 In Development
